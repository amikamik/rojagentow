  └── [Git Log shows commits from each agent]
      • ALPHA: NORMAL task completed [iteration 42]
      • BETA: HIGH priority response to ALPHA
      • GAMMA: Synthesized solution ready

  Each agent reads/writes to shared GitHub repo:
  1. git pull origin master          (get latest)
  2. Read JSON files (messages, tasks)
  3. Process highest priority
  4. run copilot -i "prompt"
  5. Write output to Markdown file
  6. git add, commit, push

┌─────────────────────────────────────────────────────────────────────────────┐
│                         KEY DESIGN DECISIONS                                 │
└─────────────────────────────────────────────────────────────────────────────┘

  ✓ Polling-based (vs event-based)
    - Simpler to implement
    - No WebSocket complexity
    - Works with GitHub's limitations
    - 5-second latency acceptable for most tasks

  ✓ JSON for communication (vs database)
    - GitHub-native (just commit files)
    - Version controlled (see all history)
    - Easy to debug (human-readable)
    - No external dependencies

  ✓ Output files for context (vs shared memory)
    - Each agent sees full history
    - Easy to trace decision-making
    - Works across agent restarts
    - GitHub backup included

  ✓ 4-level priority hierarchy
    - Prevents starvation of urgent work
    - Agents can interrupt themselves
    - Extensible (easy to add levels)
    - Clear precedence rules

  ✓ Shared agent_core.py library
    - No code duplication
    - Consistent behavior
    - Easy to add new agents
    - Centralized update location


┌─────────────────────────────────────────────────────────────────────────────┐
│                     HOW TO EXTEND THE SYSTEM                                │
└─────────────────────────────────────────────────────────────────────────────┘

  Add Agent DELTA?
  1. Add to config.json:
     "DELTA": { "role": "...", "output_file": "output/agent-delta-work.md" }

  2. Copy agent-alpha.py → agents/delta/agent-delta.py
     Change: agent = AgentCore("DELTA", repo_path)

  3. Add inbox to agent-messages.json:
     "DELTA": { "messages": [], "unread_count": 0 }

  4. Run: python agents/delta/agent-delta.py (in separate terminal)

  5. Master Control automatically works with new agent!

  Add Custom Priority Level?
  1. Edit agent_core.py select_next_prompt()
  2. Insert new priority check between existing levels
  3. Update config.json priority_levels
  4. All agents automatically use new level

  Add Webhook integration?
  1. Create webhook-listener.py
  2. Convert webhook events → agent-messages.json
  3. Agents continue working unchanged
