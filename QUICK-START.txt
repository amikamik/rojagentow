WHAT'S HAPPENING UNDER THE HOOD?
─────────────────────────────────

  Each agent runs this loop (every 5 seconds):

  1. git pull origin master
     → Get latest tasks and messages from GitHub

  2. Select highest priority prompt
     CRITICAL (user commands)
       ↓
     HIGH (from other agents)
       ↓
     NORMAL (from task queue)
       ↓
     MEDIUM (feedback)

  3. If HIGH or NORMAL: Get context from other agents
     Read their output/agent-X-work.md files

  4. Run: copilot -i "prompt [+ context from other agents]"

  5. Write results to output/agent-X-work.md
     Timestamp each entry

  6. If appropriate: Send HIGH or NORMAL priority messages
     to other agents (stored in agent-messages.json)

  7. git add -A && git commit && git push

  8. Sleep 5 seconds

  9. Loop back to step 1 (infinite)

════════════════════════════════════════════════════════════════════

PRIORITY LEVELS EXPLAINED
─────────────────────────

  🔴 CRITICAL (priority="critical")
     From: USER via Master Control
     Effect: Agent drops everything and does this immediately
     Use: "ALPHA, stop everything and analyze THIS!"

  🟠 HIGH (priority="high")
     From: Other agents OR user
     Effect: Agent responds to this before normal tasks
     Use: Agent A completed task → tells B "read my output"
     
  🟡 NORMAL (priority="normal")
     From: task-queue.json (default for user tasks)
     Effect: Normal work priority
     Use: Regular task assignments
     
  🟢 MEDIUM (priority="medium")
     From: Agent feedback/questions
     Effect: Address when there's time
     Use: "Did you consider approach X?"

TROUBLESHOOTING
───────────────

  Q: Agents don't appear in terminals?
  A: Run `python launch-control.py` → [4] Launch Agent
     Select which agent to launch

  Q: Agents started but not doing anything?
  A: Master Control → [1] Add Task
     Make sure you give them work to do!

  Q: Messages not appearing between agents?
  A: Master Control → [5] Show Agent Status
     Check unread_count
     View agent-messages.json to verify format

  Q: Agent terminal showing errors?
  A: Check in that terminal:
     - Does `copilot -i "test"` work manually?
     - Is GitHub token set? Check README-REAL-AGENTS.md

  Q: Want to stop an agent?
  A: Just close that agent's terminal window
     Agent will cleanly shutdown

  Q: Want to restart?
  A: Close all agent windows
     Run `launch-agents.bat` again

CONFIGURATION FILES

  Edit these to customize behavior!
